# 🗂️ Clutch

> Your desktop. Finally organized.

Clutch is a lightweight desktop overlay that groups your scattered
files and folders into clean visual packages. Nothing moves.
Nothing gets deleted. Just a beautiful pill-shaped dock that
keeps everything organized and one hover away.

## How It Works

1. Launch Clutch — a pill appears on your screen edge
2. Click "+" to create a package
3. Add folders and files from your desktop to the package
4. Hover the package card — see everything inside
5. Click any item — it opens directly

## Features

- Floating pill dock — always visible, never intrusive
- Colored packages — group by project, category, whatever
- Spring animations — smooth, Apple-quality feel
- Zero RAM overhead — under 5MB idle
- Nothing moves — your files stay exactly where they are
- Drag to reposition — put the pill anywhere on screen

## Tech

Built with Rust + Slint — GPU-accelerated, native performance.
No Electron. No browser. No bloat.

## Build

```bash
cargo build --release
```

Binary: `target/release/clutch.exe`
